### 1. Template Inheritance (Best Practice)

Create a base template `base.html` inside the `templates` folder:

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>{% block title %}My Flask App{% endblock %}</title>
    <style>
        body { font-family: Arial, sans-serif; margin: 40px; }
        nav { background: #333; padding: 15px; }
        nav a { color: white; margin-right: 15px; text-decoration: none; }
    </style>
</head>
<body>
    <nav>
        <a href="/">Home</a>
        <a href="/about">About</a>
        <a href="/feedback">Feedback</a>
    </nav>
    
    {% block content %}{% endblock %}
</body>
</html>
```

---

### 2. Using the Base Template

Create `home.html`:

```html
{% extends "base.html" %}

{% block title %}Home{% endblock %}

{% block content %}
    <h1>Welcome to My Website</h1>
    <p>This is the homepage using template inheritance.</p>
{% endblock %}
```

### 3. Handling Forms (POST Method)

Update `app.py`:

```python
from flask import Flask, render_template, request, redirect, url_for

app = Flask(__name__)

@app.route("/")
def home():
    return render_template("home.html")

@app.route("/feedback", methods=["GET", "POST"])
def feedback():
    if request.method == "POST":
        name = request.form.get("name")
        message = request.form.get("message")
        rating = request.form.get("rating")
        
        print(f"New Feedback - Name: {name}, Rating: {rating}, Message: {message}")
        return redirect(url_for("thank_you"))
    
    return render_template("feedback.html")

@app.route("/thank-you")
def thank_you():
    return "<h1>Thank you for your feedback! 🙏</h1>"

if __name__ == "__main__":
    app.run(debug=True)
```

### 4. Creating the Feedback Form (`feedback.html`)

```html
{% extends "base.html" %}

{% block title %}Feedback{% endblock %}

{% block content %}
    <h1>Give Us Feedback</h1>
    <form method="POST">
        <label>Name:</label><br>
        <input type="text" name="name" required><br><br>
        
        <label>Rating (1-5):</label><br>
        <input type="number" name="rating" min="1" max="5" required><br><br>
        
        <label>Your Message:</label><br>
        <textarea name="message" rows="5" cols="40" required></textarea><br><br>
        
        <button type="submit">Submit Feedback</button>
    </form>
{% endblock %}
```

---

### 📝 Notes

- Use `methods=["GET", "POST"]` for routes that handle forms.
- Access form data with `request.form.get("field_name")`.
- Always use `{% extends "base.html" %}` and blocks for consistent design.
- `url_for()` is the best way to create internal links.
