### 1. List Comprehensions

List comprehensions allow you to create new lists in a single, readable line.

```python
# Traditional way
squares = []
for i in range(1, 11):
    squares.append(i ** 2)

# Using List Comprehension (Much cleaner)
squares = [i ** 2 for i in range(1, 11)]
print(squares)
```

---

### 2. List Comprehensions with Conditions

```python
# Only even numbers
evens = [x for x in range(1, 21) if x % 2 == 0]

# Only numbers greater than 50
numbers = [10, 25, 67, 89, 34, 56]
high = [x for x in numbers if x > 50]

# With else
result = ["Even" if x % 2 == 0 else "Odd" for x in range(1, 11)]
```

---

### 3. Dictionary Comprehensions

```python
squares_dict = {x: x**2 for x in range(6)}
print(squares_dict)
# Output: {0: 0, 1: 1, 2: 4, 3: 9, 4: 16, 5: 25}
```

---

### 4. Lambda Functions

Lambda functions are small, anonymous (nameless) functions defined in one line.

```python
# Normal function
def add(a, b):
    return a + b

# Lambda version
add = lambda a, b: a + b

print(add(5, 7))        # 12
```

---

### 5. Using Lambda with Built-in Functions

```python
numbers = [5, 2, 8, 1, 9]

# Sort by value
numbers.sort(key=lambda x: x)

# Sort by last digit
numbers.sort(key=lambda x: x % 10)

# Filter using lambda
evens = list(filter(lambda x: x % 2 == 0, numbers))
```

---

### 6. Practical Example

```python
# Get names starting with 'A' and make them uppercase
names = ["alice", "bob", "anna", "david", "alex"]

a_names = [name.title() for name in names if name.startswith("a")]
print(a_names)
```

---

### 📝 Notes

- List comprehensions are fast and elegant, but don’t make them too complex.
- Lambda functions are best for short, one-time operations.
- Use them when they make your code **more readable**, not just shorter.
- `map()`, `filter()`, and `sorted()` work great with lambda.
