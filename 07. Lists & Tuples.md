### 1. What is a List?

A list is an ordered collection of items that can be changed (mutable). Lists can hold different data types.

```python
# Creating lists
fruits = ["apple", "banana", "cherry", "mango"]
numbers = [10, 20, 30, 40, 50]
mixed = ["hello", 25, 3.14, True]

print(fruits)
print(len(fruits))        # Length of the list
```

---

### 2. Accessing List Elements

```python
fruits = ["apple", "banana", "cherry", "mango"]

print(fruits[0])      # apple     (first item)
print(fruits[1])      # banana
print(fruits[-1])     # mango     (last item)
print(fruits[-2])     # cherry
```

---

### 3. List Slicing

```python
numbers = [10, 20, 30, 40, 50, 60, 70]

print(numbers[1:4])     # [20, 30, 40]
print(numbers[:3])      # [10, 20, 30]
print(numbers[4:])      # [50, 60, 70]
print(numbers[::2])     # [10, 30, 50, 70]  (every second item)
```

---

### 4. Important List Methods

```python
fruits = ["apple", "banana"]

fruits.append("orange")        # Add to the end
fruits.insert(1, "grape")      # Insert at specific position
fruits.remove("banana")        # Remove by value
popped = fruits.pop()          # Remove and return last item

print(fruits)
print("Popped item:", popped)

fruits.sort()                  # Sort alphabetically / numerically
print(fruits)
```

---

### 5. Tuples

Tuples are like lists but **immutable** (cannot be changed after creation).

```python
# Creating tuples
coordinates = (10.5, 20.8)
colors = ("red", "green", "blue")

print(coordinates[0])

# This will cause an error:
# coordinates[0] = 15
```

**When to use Tuples?**
- When data should not change (e.g., coordinates, days of week, settings).

---

### 📝 Notes

- List indices start at **0**.
- Lists are mutable, tuples are immutable.
- Use square brackets `[]` for lists, parentheses `()` for tuples.
- You can convert between them using `list()` and `tuple()`.
- Lists are one of the most commonly used data structures in Python.
