### 1. What is a Variable?

A variable is a name that refers to a value. You can think of it as a labeled box that holds data.

```python
# Creating variables
name = "Alex"
age = 20
height = 1.75
is_student = True

print(name)
print(age)
print(height)
print(is_student)
```

---

### 2. Basic Data Types in Python

| Data Type | Description              | Example                  |
|-----------|--------------------------|--------------------------|
| `int`     | Whole numbers            | `42`, `-5`, `0`          |
| `float`   | Decimal numbers          | `3.14`, `1.75`, `-0.5`   |
| `str`     | Text (strings)           | `"Hello"`, `'Python'`    |
| `bool`    | True or False            | `True`, `False`          |

---

### 3. Working with Variables

```python
# Examples of different data types
name = "Maria"          # string
age = 17                # integer
score = 92.5            # float
passed = True           # boolean

print("Name:", name)
print("Age:", age)
print("Score:", score)
print("Passed exam:", passed)
```

**Check the type of a variable:**

```python
print(type(name))    # <class 'str'>
print(type(age))     # <class 'int'>
print(type(score))   # <class 'float'>
print(type(passed))  # <class 'bool'>
```

---

### 4. Variable Naming Rules

**Good (Recommended)**
- `user_name`
- `total_score`
- `first_name`
- `is_logged_in`

**Bad / Not Allowed**
- `2name` (cannot start with number)
- `first-name` (hyphen not allowed)
- `class` (reserved keyword)
- `my name` (spaces not allowed)

**Tip**: Use `snake_case` for variable names in Python.

---

### 5. Multiple Assignment & Reassigning

```python
# Multiple assignment
x, y, z = 10, 20, 30

# Reassigning variables
age = 18
age = age + 1        # Now age is 19

print(x, y, z)
print("New age:", age)
```

---

### 📝 Notes

- Python is **dynamically typed** — you don’t need to declare the type of a variable.
- You can change the type of a variable later (reassign it).
- Strings can use single quotes `''` or double quotes `""`.
- Use meaningful names so your code is easy to read.
