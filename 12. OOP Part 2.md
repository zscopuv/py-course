### 1. What is Inheritance?

Inheritance lets a **child class** (subclass) inherit attributes and methods from a **parent class** (superclass).

```python
# Parent Class
class Animal:
    def __init__(self, name):
        self.name = name
    
    def eat(self):
        print(f"{self.name} is eating.")

# Child Class
class Dog(Animal):          # Dog inherits from Animal
    def bark(self):
        print(f"{self.name} says Woof Woof!")
    
    def eat(self):          # Method Overriding
        print(f"{self.name} is eating bones happily!")
```

---

### 2. Using the Child Class

```python
dog = Dog("Buddy")
dog.eat()      # Uses overridden method
dog.bark()
```

### 3. The `super()` Function

```python
class Student:
    def __init__(self, name, age):
        self.name = name
        self.age = age

class CollegeStudent(Student):
    def __init__(self, name, age, major):
        super().__init__(name, age)     # Call parent constructor
        self.major = major
    
    def show_info(self):
        print(f"Student: {self.name}")
        print(f"Age: {self.age}")
        print(f"Major: {self.major}")
```

---

### 4. Complete Practical Example

```python
class Vehicle:
    def __init__(self, brand, year):
        self.brand = brand
        self.year = year
    
    def info(self):
        print(f"{self.brand} - {self.year}")


class Car(Vehicle):
    def __init__(self, brand, year, doors):
        super().__init__(brand, year)
        self.doors = doors
    
    def info(self):
        super().info()                    # Call parent method
        print(f"Number of doors: {self.doors}")


class Motorcycle(Vehicle):
    def __init__(self, brand, year, has_sidecar):
        super().__init__(brand, year)
        self.has_sidecar = has_sidecar


# Usage
my_car = Car("Toyota", 2023, 4)
my_car.info()

my_bike = Motorcycle("Honda", 2022, False)
my_bike.info()
```

### 5. Key OOP Concepts Summary

| Concept           | Meaning |
|-------------------|-------|
| **Inheritance**   | Child class gets features from parent |
| **Polymorphism**  | Same method name, different behavior |
| **Overriding**    | Child class redefines parent method |
| **Encapsulation** | Protecting data (we'll cover more later) |

---

### 📝 Notes

- Use inheritance when classes have a natural “is-a” relationship (Dog is an Animal, Car is a Vehicle).
- Don’t overuse inheritance — sometimes composition is better.
- `super()` helps you call methods from the parent class cleanly.
- Class names should always use **PascalCase**.
