### 2. Using `requests` – Fetching Web Data

```python
import requests

# Get data from a public API
response = requests.get("https://jsonplaceholder.typicode.com/posts/1")

if response.status_code == 200:
    data = response.json()
    print("Title:", data["title"])
    print("Body:", data["body"])
else:
    print("Failed to fetch data")
```

---

### 3. Introduction to Pandas

Pandas is the most popular library for data analysis in Python.

```python
import pandas as pd

# Creating a DataFrame (like a table)
data = {
    "Name": ["Alice", "Bob", "Charlie", "Diana"],
    "Age": [23, 30, 25, 22],
    "City": ["London", "New York", "Paris", "Tokyo"],
    "Score": [85, 92, 78, 95]
}

df = pd.DataFrame(data)
print(df)
```

---

### 4. Basic Pandas Operations

```python
# Reading from CSV
# df = pd.read_csv("students.csv")

print(df.head())           # First 5 rows
print(df.info())           # Summary
print(df.describe())       # Statistics

# Filtering
high_scorers = df[df["Score"] > 80]
print(high_scorers)

# Sorting
sorted_df = df.sort_values(by="Score", ascending=False)
print(sorted_df)
```

---

### 5. Practical Example: Fetch and Analyze Data

```python
import requests
import pandas as pd

# Fetch user data
response = requests.get("https://jsonplaceholder.typicode.com/users")
users = response.json()

# Convert to DataFrame
df = pd.DataFrame(users)

# Clean and show important columns
print(df[["name", "email", "company"]].head())
```

---

### 📝 Notes

- Always install libraries using `pip`.
- `requests` is perfect for interacting with websites and APIs.
- `pandas` turns messy data into easy-to-use tables.
- Use `df.to_csv("output.csv", index=False)` to save data.
- You can combine everything you’ve learned so far with these libraries.
