This course takes you from **zero to confident** in Python. Each lesson is designed with clear explanations, code examples, notes, and practical exercises.

---

## 🎯 How to Use This Course

1. Open each file in order (start from Lesson 1).
2. Read the theory and code examples.
3. Complete all **Practice Exercises**.
4. Save your own practice files in the same folder.
5. Move to the next lesson only after finishing the exercises.

**Tip**: Create a subfolder called e.g. `python-course-snippets/` to save your exercise solutions.
