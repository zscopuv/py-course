### 1. What is Flask?

Flask is a **micro web framework** for Python. It is simple, flexible, and perfect for beginners who want to build web apps.

### 2. Installation

```bash

```

---

### 3. Your First Web Application

Create a new file called `app.py`:

```python
from flask import Flask

app = Flask(__name__)

@app.route("/")
def home():
    return "<h1>Hello, World! 🌍</h1><p>Welcome to my first Flask app!</p>"

@app.route("/about")
def about():
    return "<h1>About Page</h1><p>This is a simple Flask web application.</p>"

if __name__ == "__main__":
    app.run(debug=True)
```

---

### 4. How to Run the App

```bash
python app.py
```

Then open your browser and go to:  
**http://127.0.0.1:5000/**

---

### 5. Dynamic Routes

```python
@app.route("/greet/<name>")
def greet(name):
    return f"<h1>Hello, {name.capitalize()}! 👋</h1>"

@app.route("/age/<int:years>")
def show_age(years):
    return f"<h1>You are {years} years old.</h1>"
```

---

### 6. Rendering HTML Templates (Best Practice)

Create a folder called `templates` in the same directory, then create `index.html`:

```html
<!DOCTYPE html>
<html>
<head>
    <title>Home</title>
</head>
<body>
    <h1>Welcome to {{ website_name }}</h1>
    <p>Current time: {{ current_time }}</p>
</body>
</html>
```

Now update `app.py`:

```python
from flask import Flask, render_template
from datetime import datetime

app = Flask(__name__)

@app.route("/")
def home():
    return render_template("index.html", 
                         website_name="My Awesome Site",
                         current_time=datetime.now().strftime("%H:%M"))

if __name__ == "__main__":
    app.run(debug=True)
```

---

### 📝 Notes

- `debug=True` is very useful during development (shows errors on the page).
- Routes are defined using the `@app.route()` decorator.
- Use the `templates` folder for HTML files.
- Flask uses **Jinja2** as its templating engine.
