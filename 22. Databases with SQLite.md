### Welcome Back! 🚀

In this lesson, you will learn how to work with **databases** using **SQLite** — a lightweight, file-based database built into Python. Databases allow you to store data permanently and efficiently, much better than JSON or CSV for complex applications.

---

### 1. Introduction to SQLite

SQLite is perfect for beginners and intermediate developers because:
- No server installation required
- Stores everything in a single `.db` file
- Very fast and reliable

---

### 2. Basic Database Operations

```python
import sqlite3

# Connect to database (creates it if it doesn't exist)
conn = sqlite3.connect("app_database.db")
cursor = conn.cursor()

print("Database connected successfully!")
```

---

### 3. Creating a Table

```python
cursor.execute('''
    CREATE TABLE IF NOT EXISTS users (
        id INTEGER PRIMARY KEY AUTOINCREMENT,
        name TEXT NOT NULL,
        email TEXT UNIQUE,
        age INTEGER,
        created_at TEXT
    )
''')

conn.commit()
```

---

### 4. CRUD Operations (Create, Read, Update, Delete)

```python
from datetime import datetime

# CREATE - Insert data
cursor.execute('''
    INSERT INTO users (name, email, age, created_at)
    VALUES (?, ?, ?, ?)
''', ("Alex", "alex@example.com", 22, datetime.now().strftime("%Y-%m-%d")))

conn.commit()

# READ - Select data
cursor.execute("SELECT * FROM users")
users = cursor.fetchall()

for user in users:
    print(user)
```

---

### 5. Useful Operations

```python
# Update
cursor.execute("UPDATE users SET age = ? WHERE name = ?", (23, "Alex"))

# Delete
cursor.execute("DELETE FROM users WHERE id = ?", (1,))

# Search with condition
cursor.execute("SELECT * FROM users WHERE age > ?", (20,))
young_users = cursor.fetchall()
```

### 6. Complete Example: Student Database

```python
import sqlite3
from datetime import datetime

class Database:
    def __init__(self):
        self.conn = sqlite3.connect("students.db")
        self.cursor = self.conn.cursor()
        self.create_table()
    
    def create_table(self):
        self.cursor.execute('''
            CREATE TABLE IF NOT EXISTS students (
                id INTEGER PRIMARY KEY AUTOINCREMENT,
                name TEXT,
                grade TEXT,
                age INTEGER,
                enrolled_date TEXT
            )
        ''')
        self.conn.commit()
    
    def add_student(self, name, grade, age):
        self.cursor.execute('''
            INSERT INTO students (name, grade, age, enrolled_date)
            VALUES (?, ?, ?, ?)
        ''', (name, grade, age, datetime.now().strftime("%Y-%m-%d")))
        self.conn.commit()
        print(f"✅ Student '{name}' added successfully!")

    def show_all(self):
        self.cursor.execute("SELECT * FROM students")
        rows = self.cursor.fetchall()
        for row in rows:
            print(row)
```

---

### 📝 Notes

- Always call `conn.commit()` after making changes.
- Use `?` placeholders to prevent SQL injection.
- Close the connection when done: `conn.close()`
- SQLite is great for desktop apps, personal tools, and small web apps.
