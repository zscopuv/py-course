### Welcome Back! 🚀

In this lesson, you will learn how to work with dates and times in Python using the powerful `datetime` module. This is very useful for creating logs, calendars, age calculators, reminders, and many real-world applications.

---

### 1. The `datetime` Module

```python
from datetime import datetime, date, timedelta

# Get current date and time
now = datetime.now()
print(now)

# Get only current date
today = date.today()
print(today)
```

---

### 2. Formatting Dates

```python
now = datetime.now()

print(now.strftime("%Y-%m-%d"))           # 2026-05-15
print(now.strftime("%d/%m/%Y"))           # 15/05/2026
print(now.strftime("%A, %B %d, %Y"))      # Friday, May 15, 2026
print(now.strftime("%H:%M:%S"))           # 15:40:22
```

**Common Format Codes:**
- `%Y` → Year (2026)
- `%m` → Month (05)
- `%d` → Day (15)
- `%A` → Weekday name (Friday)
- `%H:%M` → Hour and Minute

---

### 3. Creating Custom Dates

```python
from datetime import datetime

birthday = datetime(2008, 12, 25)
print(birthday)

exam_date = datetime(2026, 6, 15, 9, 0)
print(exam_date)
```

---

### 4. Date Calculations

```python
from datetime import datetime, timedelta

today = datetime.now()

# Add or subtract days
one_week_later = today + timedelta(days=7)
print("One week later:", one_week_later)

yesterday = today - timedelta(days=1)
print("Yesterday was:", yesterday)

# Calculate age
birth_date = datetime(2005, 3, 10)
age = today.year - birth_date.year
print(f"You are approximately {age} years old.")
```

---

### 5. Practical Example: Days Until Event

```python
from datetime import datetime

def days_until(event_date_str):
    event_date = datetime.strptime(event_date_str, "%Y-%m-%d")
    today = datetime.now()
    difference = event_date - today
    return difference.days

print("Days until New Year:", days_until("2026-12-31"))
```

---

### 📝 Notes

- Use `datetime` for both date and time.
- Use `date` when you only need the date.
- `strftime()` = String Format Time (convert date to string)
- `strptime()` = String Parse Time (convert string to date)
- `timedelta` is used to add or subtract time.
