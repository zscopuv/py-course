**Lesson 1: Setup & Your First Python Program**

### Welcome to Python! 🎉

Python is one of the easiest and most powerful programming languages in the world. It is used for web development, data science, automation, artificial intelligence, and more.

In this first lesson, we will:
- Install Python
- Set up a code editor
- Run your very first Python program
- Understand how Python works

---

### 1. Installing Python

1. Go to the official Python website: [python.org](https://www.python.org/downloads/)
2. Download the latest version (Python 3.11 or higher).
3. **Important**: During installation, check the box that says  
   **"Add python.exe to PATH"**
4. Click "Install Now" and finish the setup.

**Verify Installation**  
Open your terminal (Command Prompt on Windows, Terminal on macOS/Linux) and type:

```bash
python --version
```

or

```bash
python3 --version
```

You should see something like `Python 3.12.x`. Great!

---

### 2. Choosing a Code Editor

**Recommended for Beginners: Visual Studio Code (VS Code)**

1. Download VS Code from: [code.visualstudio.com](https://code.visualstudio.com/)
2. Install the **Python extension** by Microsoft (it will appear automatically when you open a `.py` file).

**Alternative**: Thonny (very simple, good for absolute beginners)

---

### 3. Your First Python Program

Create a new folder called `python-course` on your desktop.

Inside it, create a new file named `hello.py`

**Write this code:**

```python
# This is my first Python program
print("Hello, World!")
print("Welcome to Python!")
print("I am ready to code 🚀")
```

---

### 4. How to Run the Program

**Method 1: Using Terminal / Command Prompt**

1. Open terminal in the folder where `hello.py` is saved.
2. Run this command:

```bash
python hello.py
```

or

```bash
python3 hello.py
```

**Method 2: Using VS Code**
- Open the file in VS Code
- Right-click inside the editor → **Run Python File in Terminal**

You should see the output:

```
Hello, World!
Welcome to Python!
I am ready to code 🚀
```

---

### Understanding the Code

```python
# This is a comment
print("Hello, World!")
```

- **`#`** → Anything after `#` is a comment (Python ignores it)
- **`print()`** → A built-in function that displays text on the screen
- **Quotes** `" "` or `' '` → Used for text (called strings)

---

### 📝 Notes

- Python is **case-sensitive** → `Print` is different from `print`
- Indentation (spaces) matters a lot in Python
- You don’t need to write `;` at the end of lines
- File names should end with `.py`
