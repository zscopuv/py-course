In this lesson, we begin **Object-Oriented Programming (OOP)** — one of the most important concepts in modern programming. OOP helps you organize code by modeling real-world things as "objects".

---

### 1. What is OOP?

OOP is a way of writing code using **Classes** and **Objects**.

- **Class** → A blueprint (template)
- **Object** → A real instance created from that blueprint

---

### 2. Creating a Class

```python
class Dog:
    """This is a class representing a dog."""
    
    # Class attribute (shared by all dogs)
    species = "Canis familiaris"
    
    # Constructor - runs when creating a new object
    def __init__(self, name, age):
        self.name = name        # Instance attribute
        self.age = age          # Instance attribute
    
    # Method (function inside a class)
    def bark(self):
        print(f"{self.name} says Woof Woof!")
    
    def describe(self):
        print(f"{self.name} is {self.age} years old.")
```

---

### 3. Creating Objects

```python
# Creating objects (instances)
dog1 = Dog("Buddy", 3)
dog2 = Dog("Luna", 5)

dog1.bark()
dog2.describe()

print(dog1.name)
print(dog2.age)
```

---

### 4. Complete Example: Student Class

```python
class Student:
    def __init__(self, name, student_id, grade):
        self.name = name
        self.student_id = student_id
        self.grade = grade
        self.courses = []
    
    def add_course(self, course):
        self.courses.append(course)
        print(f"{course} added for {self.name}")
    
    def show_info(self):
        print(f"\nStudent: {self.name}")
        print(f"ID: {self.student_id}")
        print(f"Grade: {self.grade}")
        print(f"Courses: {self.courses}")


# Using the class
student1 = Student("Alice", "S1001", "10th")
student1.add_course("Mathematics")
student1.add_course("Physics")
student1.show_info()
```

---

### 5. Important Concepts

- **`__init__`** → Special method called constructor
- **`self`** → Refers to the current object
- **Attributes** → Variables belonging to the object (`self.name`)
- **Methods** → Functions belonging to the object

---

### 📝 Notes

- Class names should use **PascalCase** (e.g., `MyClass`).
- Always include `self` as the first parameter in methods.
- OOP helps make code more organized, reusable, and easier to maintain.
- You can create as many objects as you want from one class.
