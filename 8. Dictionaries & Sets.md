### Welcome Back! 🚀

In this lesson, you will learn two very powerful Python data structures: **Dictionaries** and **Sets**.

Dictionaries let you store data as key-value pairs, while Sets are perfect for storing unique items.

---

### 1. What is a Dictionary?

A dictionary is a collection of key-value pairs. It is ordered (as of Python 3.7+), mutable, and does not allow duplicate keys.

```python
# Creating a dictionary
student = {
    "name": "Emma",
    "age": 20,
    "grade": "A",
    "courses": ["Math", "Physics"]
}

print(student)
print("Name:", student["name"])
```

---

### 2. Accessing and Modifying Dictionaries

```python
student = {"name": "Emma", "age": 20, "city": "London"}

# Accessing values
print(student["name"])

# Using .get() method (safer)
print(student.get("age"))
print(student.get("phone", "Not found"))   # Default value if key doesn't exist

# Adding / Updating
student["age"] = 21
student["phone"] = "123-456-7890"

# Removing items
student.pop("city")
del student["phone"]

print(student)
```

---

### 3. Useful Dictionary Methods

```python
student = {"name": "Emma", "age": 21, "grade": "A"}

print(student.keys())      # All keys
print(student.values())    # All values
print(student.items())     # All key-value pairs

# Looping through a dictionary
for key, value in student.items():
    print(f"{key}: {value}")
```

---

### 4. What is a Set?

A set is an unordered collection of **unique** items. Sets are mutable but cannot contain mutable elements (like lists).

```python
# Creating sets
fruits = {"apple", "banana", "cherry", "apple"}   # Duplicates are removed
print(fruits)   # {'apple', 'banana', 'cherry'}

numbers = {1, 2, 3, 4, 5}
```

---

### 5. Set Operations

```python
set1 = {1, 2, 3, 4}
set2 = {3, 4, 5, 6}

print(set1.union(set2))           # All elements from both
print(set1.intersection(set2))    # Common elements
print(set1.difference(set2))      # Elements in set1 but not in set2
print(set1.symmetric_difference(set2))  # Elements in either but not both
```

---

### 📝 Notes

- Dictionaries use curly braces `{}` with `key: value` pairs.
- Dictionary keys must be **immutable** (strings, numbers, tuples).
- Sets automatically remove duplicates.
- Sets are much faster for checking if an item exists (`in` operator).
- Use dictionaries for structured data (like records), and sets for unique collections.
