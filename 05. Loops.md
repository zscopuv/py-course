### Welcome Back! 🚀

In this lesson, you will learn how to repeat actions efficiently using **loops**. Loops are essential when you want to run the same code multiple times without writing it repeatedly.

---

### 1. The `while` Loop

The `while` loop runs as long as a condition is `True`.

```python
count = 1

while count <= 5:
    print("Count:", count)
    count = count + 1
```

**Warning**: Always make sure the condition will eventually become `False`, otherwise you create an **infinite loop**.

---

### 2. The `for` Loop

The `for` loop is commonly used to iterate over a sequence (like a range of numbers).

```python
# Using range()
for i in range(5):           # 0 to 4
    print("Number:", i)

print("---")

for i in range(1, 6):        # 1 to 5
    print("Number:", i)
```

---

### 3. Important Loop Controls

- **`break`** → Exit the loop immediately
- **`continue`** → Skip the current iteration and move to the next

```python
for number in range(1, 11):
    if number == 6:
        break                    # Stop when number is 6
    if number % 2 == 0:
        continue                 # Skip even numbers
    print(number)
```

---

### 4. Practical Examples

```python
# Example 1: Sum of first n numbers
n = int(input("Enter a number: "))
sum = 0

for i in range(1, n+1):
    sum = sum + i

print("The sum is:", sum)


# Example 2: Password attempts
attempts = 0

while attempts < 3:
    password = input("Enter password: ")
    if password == "python123":
        print("✅ Access granted!")
        break
    else:
        attempts += 1
        print(f"❌ Wrong password. {3 - attempts} attempts left.")
else:
    print("Too many failed attempts. Account locked.")
```

---

### 📝 Notes

- `range(start, stop, step)` — very useful with for loops.
- `while` is better when you don’t know how many times the loop will run.
- `for` is better when you want to repeat a known number of times.
- Indentation is critical inside loops.
- You can use `+=`, `-=`, `*=` as shortcuts (`count += 1` is same as `count = count + 1`).
