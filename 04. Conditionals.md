### 1. The `if` Statement

The `if` statement runs a block of code only if a condition is `True`.

```python
age = 18

if age >= 18:
    print("You are eligible to vote!")
    print("You can also get a driver's license.")
```

**Important**: Notice the indentation (4 spaces) after the colon `:` â€” this is required in Python.

### 3. `if` + `elif` + `else`

Use `elif` (else if) when you have multiple conditions.

```python
score = int(input("Enter your exam score (0-100): "))

if score >= 90:
    print("Grade: A - Excellent!")
elif score >= 80:
    print("Grade: B - Good job!")
elif score >= 70:
    print("Grade: C - You passed.")
elif score >= 60:
    print("Grade: D - Needs improvement.")
else:
    print("Grade: F - You failed.")
```

---

### 4. Nested Conditionals

You can put `if` statements inside other `if` statements.

```python
age = 16
has_permit = True

if age >= 18:
    print("You can drive.")
else:
    if has_permit:
        print("You can drive with a guardian.")
    else:
        print("You cannot drive yet.")
```

---

### 5. Logical Operators with Conditionals

```python
username = "admin"
password = "12345"

if username == "admin" and password == "12345":
    print("Login successful! Welcome.")
elif username == "admin" or password == "12345":
    print("One of your credentials is correct.")
else:
    print("Login failed.")
```

---

### đź“ť Notes

- Conditions must evaluate to `True` or `False`.
- Use `==` for comparison, not `=` (which is for assignment).
- `elif` and `else` are optional.
- Indentation is very important â€” wrong indentation causes errors.
- You can combine multiple conditions using `and`, `or`, and `not`.
