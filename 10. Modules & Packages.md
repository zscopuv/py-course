### Welcome Back! 🚀

In this lesson, you will learn how to use **modules** — files that contain Python code (functions, variables, classes) that you can reuse in your programs. Modules help you organize code and use powerful ready-made tools.

---

### 1. What is a Module?

A module is simply a `.py` file containing Python code. Python comes with many built-in modules.

```python
# Importing a module
import math
import random
import os
```

---

### 2. Popular Built-in Modules

#### Math Module
```python
import math

print(math.sqrt(16))      # 4.0
print(math.pi)            # 3.14159...
print(math.pow(2, 3))     # 8.0
print(math.ceil(4.3))     # 5
print(math.floor(4.7))    # 4
```

#### Random Module
```python
import random

print(random.randint(1, 100))     # Random number between 1 and 100
print(random.choice(["apple", "banana", "cherry"]))

# Shuffle a list
numbers = [1, 2, 3, 4, 5]
random.shuffle(numbers)
print(numbers)
```

#### OS Module (Operating System)
```python
import os

print(os.getcwd())           # Current working directory
print(os.listdir())          # List files in current folder

# Create a new folder
os.mkdir("new_folder")
```

---

### 3. Importing in Different Ways

```python
# Import specific functions (cleaner)
from math import sqrt, pi
print(sqrt(25))
print(pi)

# Import with alias
import datetime as dt
today = dt.datetime.now()
print(today)
```

---

### 4. Creating Your Own Module

Create a new file called `helpers.py`:

```python
# helpers.py
def greet(name):
    return f"Hello, {name}!"

def add(a, b):
    return a + b

PI = 3.14159
```

Now use it in another file:

```python
# main.py
import helpers

print(helpers.greet("Alex"))
print(helpers.add(10, 20))
print(helpers.PI)
```

---

### 5. Introduction to Packages

A package is a folder containing multiple modules and an `__init__.py` file.  
We’ll explore more advanced packages (like `pandas`) in later lessons.

---

### 📝 Notes

- Module names are usually lowercase.
- You can import modules at the top of your file.
- Creating your own modules helps keep your code clean and reusable.
- `random`, `math`, and `os` are very commonly used.
- Use `from module import function` when you only need specific tools.
