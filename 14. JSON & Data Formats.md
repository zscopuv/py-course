### 1. What is JSON?

JSON (JavaScript Object Notation) is a lightweight format for storing and exchanging data. It looks very similar to Python dictionaries.

```python
# Example JSON data
{
    "name": "Alex",
    "age": 20,
    "is_student": true,
    "hobbies": ["reading", "gaming", "coding"]
}
```

---

### 2. Working with JSON in Python

```python
import json

# Convert Python dict to JSON (Serialization)
student = {
    "name": "Alex",
    "age": 20,
    "grade": "A",
    "hobbies": ["chess", "music"]
}

# Write to JSON file
with open("student.json", "w") as file:
    json.dump(student, file, indent=4)

print("Data saved to student.json")
```

---

### 3. Reading JSON Files

```python
import json

# Read JSON file
with open("student.json", "r") as file:
    data = json.load(file)

print(data["name"])
print(data["hobbies"])
```

---

### 4. Working with CSV Files

CSV (Comma-Separated Values) is commonly used for tabular data.

```python
import csv

# Writing to CSV
with open("students.csv", "w", newline="") as file:
    writer = csv.writer(file)
    writer.writerow(["Name", "Age", "Grade"])           # Header
    writer.writerow(["Emma", 19, "A"])
    writer.writerow(["Liam", 21, "B"])
    writer.writerow(["Olivia", 20, "A"])
```

---

### 5. Reading CSV Files

```python
import csv

with open("students.csv", "r") as file:
    reader = csv.reader(file)
    next(reader)                    # Skip header
    for row in reader:
        print(f"Name: {row[0]}, Age: {row[1]}, Grade: {row[2]}")
```

---

### 6. Practical Example: Saving Game Progress

```python
import json

def save_game(player_name, level, score):
    game_data = {
        "player": player_name,
        "level": level,
        "score": score,
        "last_saved": str(datetime.now())
    }
    
    with open("save_game.json", "w") as file:
        json.dump(game_data, file, indent=4)
    
    print("Game saved successfully!")

save_game("DragonSlayer", 15, 2450)
```

---

### 📝 Notes

- `json.dump()` → Write Python object to JSON file
- `json.load()` → Read JSON file into Python dictionary
- Always use `indent=4` for readable JSON files
- CSV is great for spreadsheet-like data
- `newline=""` is recommended when writing CSV files
