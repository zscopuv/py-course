### 1. Arithmetic Operators

These operators are used to perform mathematical operations.

| Operator | Description          | Example          | Result |
|----------|----------------------|------------------|--------|
| `+`      | Addition             | `5 + 3`          | `8`    |
| `-`      | Subtraction          | `10 - 4`         | `6`    |
| `*`      | Multiplication       | `6 * 7`          | `42`   |
| `/`      | Division             | `15 / 2`         | `7.5`  |
| `//`     | Floor Division       | `15 // 2`        | `7`    |
| `%`      | Modulus (remainder)  | `15 % 2`         | `1`    |
| `**`     | Exponentiation       | `2 ** 3`         | `8`    |

```python
a = 10
b = 3

print(a + b)   # 13
print(a - b)   # 7
print(a * b)   # 30
print(a / b)   # 3.333...
print(a // b)  # 3
print(a % b)   # 1
print(a ** b)  # 1000
```

---

### 2. Comparison Operators

Used to compare values. They return `True` or `False`.

| Operator | Meaning                  |
|----------|--------------------------|
| `==`     | Equal to                 |
| `!=`     | Not equal to             |
| `>`      | Greater than             |
| `<`      | Less than                |
| `>=`     | Greater than or equal    |
| `<=`     | Less than or equal       |

```python
age = 18
print(age >= 18)   # True
print(age == 21)   # False
```

---

### 3. Logical Operators

| Operator | Description |
|----------|-----------|
| `and`    | True if both conditions are true |
| `or`     | True if at least one is true     |
| `not`    | Reverses the result              |

```python
has_ticket = True
is_adult = True

print(has_ticket and is_adult)   # True
print(has_ticket or is_adult)    # True
print(not has_ticket)            # False
```

---

### 4. User Input

Use the `input()` function to get data from the user.

```python
name = input("Enter your name: ")
print("Hello,", name)

age = input("Enter your age: ")
print("You are", age, "years old")
```

**Important**: `input()` always returns a **string**. You need to convert it if you want numbers.

---

### 5. Type Conversion

```python
# Convert string to integer
age = int(input("Enter your age: "))

# Convert string to float
height = float(input("Enter your height in meters: "))

print("Next year you will be", age + 1)
print("Your height in cm:", height * 100)
```

---

### 📝 Notes

- Use meaningful variable names.
- Always convert user input when doing math (`int()` or `float()`).
- Division `/` always returns a float.
- Floor division `//` removes the decimal part.
- You can combine operators in one line (e.g., `total = a + b * c`).
