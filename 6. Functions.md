### Welcome Back! 🚀

In this lesson, you will learn how to write **functions** — reusable blocks of code that help you organize your program and avoid repeating the same code.

Functions are one of the most important concepts in programming.

---

### 1. What is a Function?

A function is a named block of code that performs a specific task. You can call (use) it whenever you need.

```python
# Defining a function
def greet():
    print("Hello, welcome to Python!")
    print("I hope you're having a great day!")


# Calling the function
greet()
greet()
```

---

### 2. Functions with Parameters

Parameters allow you to pass information into a function.

```python
def greet(name):
    print(f"Hello, {name}!")
    print("Welcome to Lesson 6\n")


greet("Alice")
greet("Bob")
greet("Maria")
```

---

### 3. Functions with Return Values

Use the `return` keyword to send a value back from the function.

```python
def add(a, b):
    return a + b


result = add(10, 25)
print("Sum is:", result)

# You can also use it directly
print("Total:", add(50, 30))
```

---

### 4. Default Parameters

```python
def introduce(name, age=18):
    print(f"My name is {name} and I am {age} years old.")


introduce("David")           # Uses default age 18
introduce("Emma", 22)        # Uses provided age
```

---

### 5. Multiple Parameters & Scope

```python
def calculate_bmi(weight, height):
    bmi = weight / (height ** 2)
    return round(bmi, 2)


bmi_value = calculate_bmi(70, 1.75)
print("Your BMI is:", bmi_value)
```

---

### 📝 Notes

- Function names should be lowercase with underscores (`snake_case`).
- Always define a function before calling it.
- Variables created inside a function are local (they don’t exist outside).
- Use `return` when you need the function to give back a result.
- Functions make your code cleaner, reusable, and easier to debug.
